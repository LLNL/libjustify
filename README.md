libjustify
==========

### Authors

libjustify was created by Barry Rountree (rountree@llnl.gov).

### Installing

Installation is as simple as:
```
git clone https://github.com/LLNL/libjustify.git
mkdir build && cd build
cmake ..
cmake --build .
```
